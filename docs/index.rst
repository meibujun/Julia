Welcome to GenoStockPy's documentation!
========================================

GenoStockPy is a Python library for quantitative genetics and genomics analyses,
including mixed model equation solving, MCMC-based variance component estimation,
genomic prediction (GBLUP, SSGBLUP, Bayesian Alphabet), and Genome-Wide
Association Studies (GWAS).

This documentation provides an overview of the library, installation instructions,
tutorials, and the API reference.
